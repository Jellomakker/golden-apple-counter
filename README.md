# Golden Apple Counter

Client-side Fabric mod for Minecraft `1.21.5+` that tracks how many golden apples each opponent has eaten.

When a tracked player has eaten at least one apple, their overhead name is shown as:

- `ğŸ <count> <player name>`

## Build

```bash
./gradlew build
```

Built jar:

- `build/libs/golden-apple-counter-1.0.0+mc1.21.5.jar`

## Loader/API

- Fabric Loader `0.18.4+`
- Fabric API `0.128.2+1.21.5`
- Java `21+`
